abstract  = {
Efficient graph partitioning is a key factor affecting the performance and scalability of distributed graph databases. The placement of graph vertices across multiple storage nodes determines the amount of inter-shard communication required during graph traversal and shortest-path queries. Modern partitoning algorithms can be roughly devided in two groups: offline and online, in which former take significantly more time than the latter but provide better results. This paper presents a hybrid graph partitioning approach that combines the online partitioning of Fennel algorithm and refintment approach used in METIS framework. The proposed method first constructs an initial balanced partitioning using Fennel and subsequently applies KL-based local optimization to reduce the number of edges crossing shard boundaries while preserving partition balance. This combination leverages the scalability of streaming partitioning and the optimization capabilities of iterative local refinement.
