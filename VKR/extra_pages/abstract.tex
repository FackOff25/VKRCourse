\section*{Аннотация}
В работе рассматривается задача распределения вершин в распределённой графовой базе данных. Актуальность исследования обусловлена ростом объёмов графовых данных и необходимостью повышения эффективности их хранения и обработки в распределённых системах.

Проведён анализ существующих методов распределения графов, включая структурные, учитывающие рабочую нагрузку и потоковые подходы. На основе проведённого исследования разработан программный модуль распределения вершин графа между шардами и реализованы средства оценки качества получаемого распределения.

Эффективность разработанного решения оценивалась по количеству межшардовых переходов, возникающих при выполнении запросов поиска пути. Результаты экспериментов показали, что использование специализированных алгоритмов распределения позволяет существенно снизить количество межшардовых взаимодействий по сравнению со отдельными существующими решениями.
\section*{Abstract}
This work focuses on the problem of vertex distribution in a distributed graph database. The relevance of the study is driven by the growing volume of graph data and the need to improve the efficiency of distributed data storage and processing systems.

Existing graph partitioning approaches, including structure-based, workload-aware, and streaming methods, were analyzed. Based on the results of this analysis, a software module for graph vertex distribution was developed. The module enables vertex placement across shards and provides tools for evaluating the quality of the resulting distribution.

The effectiveness of the proposed solution was assessed using the number of inter-shard transitions generated during path-finding queries. Experimental results showed that specialized partitioning algorithms significantly reduce inter-shard interactions compared to random data placement.
