\section{Анализ методов распределения графовых данных}

Задача распределения графовых данных заключается в разбиении множества вершин графа на несколько непересекающихся частей (шардов) таким образом, чтобы обеспечить эффективное выполнение запросов над данными в распределённой среде. Основной целью является минимизация стоимости обработки рабочей нагрузки, которая, как правило, выражается через количество мажшардовых переходов при выполнении запросов, при одновременном соблюдении ограничений на балансировку размеров партиций.

Для выполнения работы были проанализированы методы из трёх следующих групп:
\begin{enumerate}
    \item Методы распределения вершин графа по хранилищам исходя только из структуры графа (методы структурной оптимизации распределения).
    \item Методы распределения вершин графа по хранилищам исходя из структуры графа и данных о запросах (методы структурной оптимизации распределения с учётом нагрузки).
    \item Методы распределения новых вершин в уже распределённый граф (методы потокового распределения).
\end{enumerate}
